# ПРАКТИЧЕСКИЕ ЗАДАНИЯ
## Технологии DevOps в процессе разработки программного обеспечения

### Предисловие
В рамках курса будут рассмотрены наиболее популярные DevOps-практики и инструменты.<br>
Программа направлена на то, чтобы вы не только узнали о них, но и получили первый практический опыт их применения.<br>
Учебное время ограничено, поэтому детальное изучение каждого инструмента выходит за рамки курса.<br>
Тем не менее, вы получите базовые знания, в которые при желании сможете самостоятельно углубиться в дальнейшем.
